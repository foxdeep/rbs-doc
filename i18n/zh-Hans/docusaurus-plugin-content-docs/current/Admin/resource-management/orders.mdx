订单页面允许管理员查看和管理所有资源订单。每个订单包括详细信息，例如其状态、唯一 ID、创建时间、申请人、预定时间和总金额。

### 主要功能

- **搜索与筛选**：使用搜索栏快速定位特定订单，或应用筛选器以缩小结果范围。
- **分页功能**：使用分页控制高效浏览订单。
- **订单详细信息**：查看每个订单的详细信息，包括其当前状态和相关元数据。
- **导出**：将订单列表导出为 Excel 文件，用于离线分析和报告。

### 使用方法

1. 导航到侧边栏中的 **资源管理** 部分。
2. 选择 **订单** 标签以查看所有订单列表。
3. 使用搜索栏或筛选器查找特定订单。
4. 点击订单以查看或编辑其详细信息。

此页面对于有效跟踪和管理资源分配至关重要。
