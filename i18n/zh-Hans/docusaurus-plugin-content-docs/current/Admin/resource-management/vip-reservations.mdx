VIP 预约功能允许管理员管理和追踪 VIP 客户的预约。这包括查看预约的详细信息，例如状态、ID、项目、预约时间、申请人和电子邮件地址。此外，管理员还可以为 VIP 客户创建新的预约，并将预约列表导出为 Excel 文件。

### 主要功能

- **预约状态**：轻松识别预约的当前状态（例如已预订、待处理或已取消）。
- **详细信息**：查看关键详细信息，例如预约 ID、项目类型、时间段和申请人信息。
- **搜索与筛选**：使用搜索栏快速定位特定预约，或应用筛选器以获得更精确的结果。
- **分页功能**：轻松浏览多页的预约记录。
- **创建预约**：管理员可以直接从系统为 VIP 客户创建新的预约。

  ![snapshot](/assets/admin/resource-management-vip-reservation-add.png)

- **导出为 Excel**：将预约列表导出为 Excel 文件，用于报告或记录保存。

此功能确保了 VIP 预约的高效管理，为管理员和客户提供了无缝的使用体验。
