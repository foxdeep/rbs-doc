**预约**功能允许管理员高效地管理和追踪资源预约。它提供了两种视图，**日历**和**表格**，以便有效地可视化和管理预约。

### 主要功能

- **📅 日历视图**：以清晰、组织化的格式显示预约，方便识别可用时段并避免排程冲突。日历视图支持多种子视图：
  - **月视图**：提供整个月份的预约概览。
  - **周视图**：显示特定周的预约，提供更详细的排程。
  - **日视图**：专注于单日的预约，适合精确管理。
  - **列表视图**：以时间顺序显示预约，方便快速参考。
- **📋 表格视图**：以表格格式列出预约，支持快速排序、筛选和详细管理。管理员还可以将预约列表导出为 Excel 文件，用于报告或离线使用。
- **⚡ 实时更新**：即时反映更改，确保排程准确。
- **🔍 预约详情**：查看和管理特定预约信息。

### 使用方法

1. 导航到侧边栏中的 **资源管理** 部分。
2. 选择 **预约** 标签以访问可用视图。
3. 在 **日历视图** 和 **表格视图** 之间切换，以根据需要管理预约。
   - 在 **日历视图** 中，切换 **月视图**、**周视图**、**日视图** 和 **列表视图** 以获取不同层次的详细信息。
4. 在 **表格视图** 中，使用导出选项下载预约列表为 Excel 文件。
5. 点击日期或行以查看或新增当日的预约。
6. 使用筛选器或搜索功能定位特定预约。

此工具旨在简化预约流程并提高资源利用率。
