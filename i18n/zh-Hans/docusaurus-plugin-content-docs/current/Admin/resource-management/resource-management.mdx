资源管理是 RBS 管理员入口网站中管理组织资源的核心中心。高效地新增、编辑、组织和优化资源分配，以消除冲突并最大化利用率。

## 功能

![snapshot](/assets/admin/resource-management-card.png)

| 功能         | 描述                                             |
| ------------ | ------------------------------------------------ |
| 🛠️ 品项      | 管理与资源相关的品项（新增、查看、更新、删除）。 |
| 📦 订单      | 查看和管理与资源相关的订单。                     |
| 📅 预约      | 监督和管理资源预约。                             |
| ⭐ VIP 预约  | 管理 VIP 用户的专属预约。                        |
| 🚦 上架/下架 | 控制资源的可用性。                               |
| ✏️ 编辑      | 修改资源详细信息。                               |
| 🗑️ 删除      | 从系统中移除资源。                               |

这些功能提供了一套全面的工具，用于有效管理资源，确保资源被高效利用并满足组织需求。

## 详细信息

### 基本信息

![snapshot](/assets/admin/resource-management-basic.png)

在“基本信息”部分配置资源的基本属性：

- 📌 **类型**：固定（例如停车场）或非固定（例如餐饮预订）。
- 📝 **名称**：资源的名称，便于识别。
- 📢 **说明**：显示与资源相关的重要信息。
- 🖼️ **图片**：提供资源的主视觉。
- 📂 **分类**：分类和组织资源。

这些设置简化了资源识别并提升了用户体验。

### 时段

![snapshot](/assets/admin/resource-management-timePeriod.png)

“时段”部分允许管理员有效地定义和管理资源的可用性。主要功能包括：

- 📅 **开放日期范围**：指定资源可用的日期范围。如果未设置范围，资源将无限期可用。
- 🚫 **关闭周末**：轻松配置资源在周末不可用。

#### 高级设置

![snapshot](/assets/admin/resource-management-advanced.png)

“高级设置”部分提供额外的配置选项，以微调资源管理。主要功能包括：

- **📆 当日预订**：限制用户提前多长时间可以预订资源，以确保公平访问并防止超额预订。
- **⏳ 限制提前预订**：设置预订的最短提前时间，让管理员能够有效准备资源。
- **📅 仅显示几天**：配置预订界面中显示的天数数量，提供清晰简洁的用户体验。
- **🔒 隔日强制关闭时段**：定义资源不可用的特定时段，例如维护。
- **📂 排除法定假日**：自动从资源的可用性中移除法定假日，简化排程。
- **📅 额外天数**：新增额外的开放或关闭天数，以根据需要自定义资源的时间表。

这些高级设置使管理员能够对资源可用性实施精确控制，确保高效和有效的管理。

### 结单

![snapshot](/assets/admin/resource-management-settlement.png)

“结单”部分允许管理员配置如何完成和处理预订。当达到指定的结算时间时，系统会自动将预订详细信息发送到管理员的待办事项清单中进行审核和处理。这确保所有预订都被记录并有效处理。

主要功能包括：

- **⏰ 每日结算时间**：管理员可以设置每日结算的具体时间。在此时间，所有资源的预订将被汇总并发送到管理员的待办事项清单。
- **🔄 自动化流程**：系统自动化结算流程，减少手动工作量并确保预订管理的准确性。

此功能简化了预订工作流程，为管理员提供清晰有序的方式来管理资源使用并确保责任分明。

### 流程

![snapshot](/assets/admin/resource-management-process.png)

“流程”部分定义了签核和经办流程，确保与资源相关的任务得到顺畅高效的管理。经办流程与结算时间密切相关，会在指定的每日结算时间自动将任务发送到管理员的待办事项清单中进行审核和处理。

#### 签核

![snapshot](/assets/admin/resource-management-approver.png)

管理员可以配置签核工作流程，确保资源预订或更改在最终确定之前得到审核和授权。主要功能包括：

- **🔄 启用签核工作流程**：为特定资源启用签核流程，要求指定的签核人审核和批准请求。
- **➕ 新增工作流程**：通过指定签核人、签核层级和触发条件来定义自定义签核工作流程。

#### 经办

![snapshot](/assets/admin/resource-management-deliver.png)

经办流程专为处理人员设计，确保与资源预订相关的任务得到系统化管理。一旦订单创建，将分配给指定的处理人员。只有在处理人员处理任务后，预订才会最终确定。主要功能包括：

- **🔄 启用经办工作流程**：为特定资源启用经办流程，要求指定的处理人员审核并分配资源。
- **➕ 可自定义工作流程**：定义触发条件和操作，以量身定制任务经办流程，确保任务分配给适当的处理人员或团队。

此流程确保资源管理任务得到高效处理，减少手动工作量，并确保预订在适当审核和处理后完成。

### 限制

![snapshot](/assets/admin/resource-management-restrictions.png)

“限制”部分使管理员能够为资源使用设置特定规则和限制，确保符合组织政策并防止滥用。主要功能包括：

- **👤 身份限制**：定义哪些用户群组可以访问资源。选项包括 RBS 账户、SSO 账户、访客或 APP-ID 验证。这确保只有授权用户可以预订或使用资源。

- **🔒 购买限制**：限制用户在定义的范围内只能购买资源一次，防止重复预订或滥用。

- **⏳ 同时预订限制**：限制用户一次只能预订一种类型的资源，确保公平访问并避免超额预订。

这些限制为管理员提供了灵活性，以执行政策并保持对资源分配的控制，确保资源被高效且公平地使用。

### 其他

![snapshot](/assets/admin/resource-management-others.png)

“其他”部分提供了额外的配置选项，以增强资源管理的灵活性和功能。这些设置允许管理员进一步自定义资源体验，确保其符合组织需求。主要功能包括：

- **📧 电子邮件通知**：配置电子邮件提醒，用于通知用户其预订情况。

- **🔔 弹出公告**：启用弹出消息，在用户与资源互动时显示重要公告或提醒。这确保重要信息得到有效传达。

- **🛸 扩展字段**：从设置中导入自定义字段，以分配特定位置或资源的其他属性，为用户提供清晰准确的信息，符合组织需求。

这些选项使管理员能够提供无缝且用户友好的体验，同时保持对资源通信和可访问性的控制。
