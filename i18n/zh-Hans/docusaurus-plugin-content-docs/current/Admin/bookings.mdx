管理员可以在「预订记录」部分轻松存取有关所有预订资源的详细资讯。此功能提供了一个有组织的预订清单，可以快速方便地存取基本详细信息，例如预订的资源、日期、时间和用户/顾客资讯。允许管理员有效地监控资源利用率。

## 功能

- 📁 过滤

  为了进一步增强报表功能，预订记录部分提供了过滤选项。管理员可以套用各种筛选器来缩小报告资料的范围。这可能包括按资源类型和日期范围进行过滤。这些过滤器可协助管理员提取相关见解，并根据对组织最重要的标准做出数据驱动的决策。

- 📊 汇出报告

  预订记录部分的突出功能之一是汇出报告。管理员可以产生包含预订资讯的报告，这对于追踪资源使用情况、评估需求和做出明智的决策非常宝贵。这些报告包含特定的资料字段，使管理员能够专注于最重要的资讯。
