顾客（员工）后台中的「我的订单」部分是您浏览所有预订的入口。在这里，您可以轻松存取有关预订的详细资讯、查看预订详细资讯并根据需要采取特定操作。

- 🗄️ 查看所有预订

  在“我的订单”中，您可以查看所有预订的完整列表，无论是过去的、当前的还是即将到来的预订。每次预订都会显示重要讯息，包括资源、日期、时间以及与预订相关的任何其他详细资讯。

- 🔍 过滤和搜寻

  为了简化您的预订记录，「我的订单」部分可让您过滤和搜寻特定预订。您可以使用日期范围或预订编号缩小范围，确保您可以快速找到您有兴趣的预订。

- 🖊️ 管理您的预订

  对于每个预订，您都可以选择透过 <span class="icon">more_vert</span> 按钮执行特定操作。以下是两个选项：

- ❌ 取消预订

  如果您需要取消预订，可以直接从选单中取消。此功能为您的预订提供了灵活性和控制。

- 👀 查看订单列表

  选择此选项，您可以查看也预订了相同资源的其他人的清单。此资讯对于与使用相同资产的其他顾客（员工）进行协作或协调非常有价值。
