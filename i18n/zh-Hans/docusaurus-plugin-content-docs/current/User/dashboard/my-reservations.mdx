「我的预订」是用户仪表板中的一个功能，提供动态且互动式的日历视图，展示您所有的预订。此功能旨在为您提供清晰且有条理的预订概览，让您能够有效地管理行程。

- 📅 日历视图

  日历提供直观且有条理的方式来查看您的预订。每个预订都显示在相应的日期上，方便您追踪和管理行程。

- 🔄 切换视图

  为了满足不同的偏好，日历支持多种视图，例如月视图、周视图、日视图和列表布局。您可以轻松切换这些视图，以获得更广泛或更详细的预订概览。

- 🔍 按日期范围筛选

  日历包含日期范围筛选功能，允许您将视图缩小到特定的时间范围内。这确保您能快速专注于选定范围内的预订。

- 🖊️ 管理您的预订

  日历上的每个预订都是互动式的，您可以直接从日历界面执行特定操作。例如：

  - ❌ 取消预订

    如果您需要取消预订，只需点击该预订并选择取消选项。这提供了一种快速且无忧的方式来管理行程变更。

  - 📋 查看详情

    点击预订还可以查看该预订的详细信息，包括资源、时间和任何附加备注。

通过这些功能，「我的预订」日历确保了管理您的预订既直观又高效。
