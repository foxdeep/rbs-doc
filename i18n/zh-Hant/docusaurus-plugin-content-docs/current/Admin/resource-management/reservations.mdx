**預約**功能允許管理員高效地管理和追蹤資源預訂。它提供了兩種視圖，**日曆**和**表格**，以便有效地可視化和管理預約。

### 主要功能

- **📅 日曆視圖**：以清晰、組織化的格式顯示預約，方便識別可用時段並避免排程衝突。日曆視圖支持多種子視圖：
  - **月視圖**：提供整個月的預約概覽。
  - **週視圖**：顯示特定週的預約，提供更詳細的排程。
  - **日視圖**：專注於單日的預約，適合精確管理。
  - **列表視圖**：以時間順序顯示預約，方便快速參考。
- **📋 表格視圖**：以表格格式列出預約，支持快速排序、篩選和詳細管理。管理員還可以將預約列表匯出為 Excel 文件，用於報告或離線使用。
- **⚡ 實時更新**：即時反映更改，確保排程準確。
- **🔍 預約詳情**：查看和管理特定預約資訊。

### 使用方法

1. 導航到側邊欄中的 **資源管理** 部分。
2. 選擇 **預約** 標籤以訪問可用視圖。
3. 在 **日曆視圖** 和 **表格視圖** 之間切換，以根據需要管理預約。
   - 在 **日曆視圖** 中，切換 **月視圖**、**週視圖**、**日視圖** 和 **列表視圖** 以獲取不同層次的詳細資訊。
4. 在 **表格視圖** 中，使用匯出選項下載預約列表為 Excel 文件。
5. 點擊日期或行以查看或新增當日的預約。
6. 使用篩選器或搜尋功能定位特定預約。

此工具旨在簡化預約流程並提高資源利用率。
