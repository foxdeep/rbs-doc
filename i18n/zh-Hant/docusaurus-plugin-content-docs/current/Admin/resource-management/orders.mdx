訂單頁面允許管理員查看和管理所有資源訂單。每個訂單包括詳細資訊，例如其狀態、唯一 ID、創建時間、申請人、預定時間和總金額。

### 主要功能

- **搜尋與篩選**：使用搜尋欄快速定位特定訂單，或應用篩選器以縮小結果範圍。
- **分頁功能**：使用分頁控制高效瀏覽訂單。
- **訂單詳細資訊**：查看每個訂單的詳細資訊，包括其當前狀態和相關元數據。
- **匯出**：將訂單列表匯出為 Excel 文件，用於離線分析和報告。

### 使用方法

1. 導航到側邊欄中的 **資源管理** 部分。
2. 選擇 **訂單** 標籤以查看所有訂單列表。
3. 使用搜尋欄或篩選器查找特定訂單。
4. 點擊訂單以查看或編輯其詳細資訊。

此頁面對於有效跟蹤和管理資源分配至關重要。
