資源管理是 RBS 管理員入口網站中管理組織資源的核心中心。高效地新增、編輯、組織和優化資源分配，以消除衝突並最大化利用率。

---

## 功能

![snapshot](/assets/admin/resource-management-card.png)

| 功能         | 描述                                                 |
| ------------ | ---------------------------------------------------- |
| 🛠️ 品項      | 管理與資源相關的個別項目（新增、查看、更新、刪除）。 |
| 📦 訂單      | 查看和管理與資源相關的訂單。                         |
| 📅 預約      | 監督和管理資源預約。                                 |
| ⭐ VIP 預約  | 管理 VIP 用戶的專屬預約。                            |
| 🚦 上架/下架 | 控制資源的可用性。                                   |
| ✏️ 編輯      | 修改資源詳細資訊。                                   |
| 🗑️ 刪除      | 從系統中移除資源。                                   |

這些功能提供了一套全面的工具，用於有效管理資源，確保資源被高效利用並滿足組織需求。

## 詳細資訊

### 基本資訊

![snapshot](/assets/admin/resource-management-basic.png)

在「基本資訊」部分配置資源的基本屬性：

- 📌 **類型**：固定（例如停車場）或非固定（例如餐點預訂）。
- 📝 **名稱**：資源的名稱，便於識別。
- 📢 **說明**：顯示與資源相關的重要訊息。
- 🖼️ **圖片**：提供資源的主視覺。
- 📂 **分類**：分類和組織資源。

這些設置簡化了資源識別並提升了用戶體驗。

### 時段

![snapshot](/assets/admin/resource-management-timePeriod.png)

「時段」部分允許管理員有效地定義和管理資源的可用性。主要功能包括：

- 📅 **開放日期範圍**：指定資源可用的日期範圍。如果未設置範圍，資源將無限期可用。
- 🚫 **關閉週末**：輕鬆配置資源在週末不可用。

#### 進階設定

![snapshot](/assets/admin/resource-management-advanced.png)

「進階設定」部分提供額外的配置選項，以微調資源管理。主要功能包括：

- **📆 當日預訂**：限制用戶提前多長時間可以預訂資源，以確保公平訪問並防止超額預訂。
- **⏳ 限制提前預訂**：設置預訂的最短提前時間，讓管理員能夠有效準備資源。
- **📅 僅顯示幾天**：配置預訂介面中顯示的天數數量，提供清晰簡潔的用戶體驗。
- **🔒 隔日強制關閉時段**：定義資源不可用的特定時段，例如維護。
- **📂 排除國定假日**：自動從資源的可用性中移除國定假日，簡化排程。
- **📅 額外天數**：新增額外的開放或關閉天數，以根據需要自訂資源的時間表。

這些進階設定使管理員能夠對資源可用性實施精確控制，確保高效和有效的管理。

### 結單

![snapshot](/assets/admin/resource-management-settlement.png)

「結單」部分允許管理員配置如何完成和處理預訂。當達到指定的結算時間時，系統會自動將預訂詳細資訊發送到管理員的待辦事項清單中進行審核和處理。這確保所有預訂都被記錄並有效處理。

主要功能包括：

- **⏰ 每日結算時間**：管理員可以設置每日結算的具體時間。在此時間，所有資源的預訂將被匯總並發送到管理員的待辦事項清單。
- **🔄 自動化流程**：系統自動化結算流程，減少手動工作量並確保預訂管理的準確性。

此功能簡化了預訂工作流程，為管理員提供清晰有序的方式來管理資源使用並確保責任分明。

### 流程

![snapshot](/assets/admin/resource-management-process.png)

「流程」部分定義了簽核和經辦流程，確保與資源相關的任務得到順暢高效的管理。經辦流程與結算時間密切相關，會在指定的每日結算時間自動將任務發送到管理員的待辦事項清單中進行審核和處理。

#### 簽核

![snapshot](/assets/admin/resource-management-approver.png)

管理員可以配置簽核工作流程，確保資源預訂或更改在最終確定之前得到審核和授權。主要功能包括：

- **🔄 啟用簽核工作流程**：為特定資源啟用簽核流程，要求指定的簽核人審核和批准請求。
- **➕ 新增工作流程**：通過指定簽核人、簽核層級和觸發條件來定義自訂簽核工作流程。

#### 經辦

![snapshot](/assets/admin/resource-management-deliver.png)

經辦流程專為處理人員設計，確保與資源預訂相關的任務得到系統化管理。一旦訂單創建，將分配給指定的處理人員。只有在處理人員處理任務後，預訂才會最終確定。主要功能包括：

- **🔄 啟用經辦工作流程**：為特定資源啟用經辦流程，要求指定的處理人員審核並分配資源。
- **➕ 可自訂工作流程**：定義觸發條件和操作，以量身定制任務經辦流程，確保任務分配給適當的處理人員或團隊。

此流程確保資源管理任務得到高效處理，減少手動工作量，並確保預訂在適當審核和處理後完成。

### 限制

![snapshot](/assets/admin/resource-management-restrictions.png)

「限制」部分使管理員能夠為資源使用設置特定規則和限制，確保符合組織政策並防止濫用。主要功能包括：

- **👤 身份限制**：定義哪些用戶群組可以訪問資源。選項包括 RBS 帳戶、SSO 帳戶、訪客或 APP-ID 驗證。這確保只有授權用戶可以預訂或使用資源。

- **🔒 購買限制**：限制用戶在定義的範圍內只能購買資源一次，防止重複預訂或濫用。

- **⏳ 同時預訂限制**：限制用戶一次只能預訂一種類型的資源，確保公平訪問並避免超額預訂。

這些限制為管理員提供了靈活性，以執行政策並保持對資源分配的控制，確保資源被高效且公平地使用。

### 其他

![snapshot](/assets/admin/resource-management-others.png)

「其他」部分提供了額外的配置選項，以增強資源管理的靈活性和功能。這些設置允許管理員進一步自訂資源體驗，確保其符合組織需求。主要功能包括：

- **📧 電子郵件通知**：配置電子郵件提醒，用於通知用戶其預訂情況。

- **🔔 彈出公告**：啟用彈出訊息，在用戶與資源互動時顯示重要公告或提醒。這確保重要資訊得到有效傳達。

- **🛸 擴展欄位**：從設置中導入自訂欄位，以分配特定位置或資源的其他屬性，為用戶提供清晰準確的資訊，符合組織需求。

這些選項使管理員能夠提供無縫且用戶友好的體驗，同時保持對資源通信和可訪問性的控制。
