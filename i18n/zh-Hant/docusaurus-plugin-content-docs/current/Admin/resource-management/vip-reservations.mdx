VIP 預約功能允許管理員管理和追蹤 VIP 客戶的預約。這包括查看預約的詳細資訊，例如狀態、ID、項目、預約時間、申請人和電子郵件地址。此外，管理員還可以為 VIP 客戶創建新的預約，並將預約列表匯出為 Excel 文件。

### 主要功能

- **預約狀態**：輕鬆識別預約的當前狀態（例如已預訂、待處理或已取消）。
- **詳細資訊**：查看關鍵詳細資訊，例如預約 ID、項目類型、時間段和申請人資訊。
- **搜尋與篩選**：使用搜尋欄快速定位特定預約，或應用篩選器以獲得更精確的結果。
- **分頁功能**：輕鬆瀏覽多頁的預約記錄。
- **創建預約**：管理員可以直接從系統為 VIP 客戶創建新的預約。

  ![snapshot](/assets/admin/resource-management-vip-reservation-add.png)

- **匯出為 Excel**：將預約列表匯出為 Excel 文件，用於報告或記錄保存。

此功能確保了 VIP 預約的高效管理，為管理員和客戶提供了無縫的使用體驗。
