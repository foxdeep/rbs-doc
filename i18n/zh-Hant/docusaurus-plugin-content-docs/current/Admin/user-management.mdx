## 目前使用者

「個人資訊」對話方塊允許使用者修改其個人資訊，例如：

- 使用者名稱
- 電子郵件地址
- 密碼

## 使用者管理

「使用者管理」頁面使用戶能夠新增使用者作為管理員或一般使用者。

管理員擁有對各種功能執行 CRUD（建立、讀取、更新、刪除）操作的完全授權。

一般使用者只有讀取資訊的權限。

<BrowserWindow url={url}>

![snapshot](/assets/admin/user-management.png)

</BrowserWindow>


### 新增

要建立用戶，需要以下欄位：

- 使用者名稱
- 電子郵件地址
- 密碼

### 編輯

當使用者點擊「更多」選單中的「編輯」時，將在對話方塊中顯示相關使用者的可編輯資訊。所有顯示的欄位都可以編輯。

### 刪除

當使用者點擊「更多」選單中的「刪除」時，將顯示一個警告對話框，以仔細檢查並提醒使用者刪除操作是不可逆的。
