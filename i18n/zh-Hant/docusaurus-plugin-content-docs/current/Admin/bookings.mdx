管理員可以在「預訂記錄」部分輕鬆存取有關所有預訂資源的詳細資訊。此功能提供了一個有組織的預訂清單，可以快速方便地存取基本詳細信息，例如預訂的資源、日期、時間和用戶/顧客資訊。允許管理員有效地監控資源利用率。

## 功能

- 📁 過濾

  為了進一步增強報表功能，預訂記錄部分提供了過濾選項。管理員可以套用各種篩選器來縮小報告資料的範圍。這可能包括按資源類型和日期範圍進行過濾。這些過濾器可協助管理員提取相關見解，並根據對組織最重要的標準做出數據驅動的決策。

- 📊 匯出報告

  預訂記錄部分的突出功能之一是匯出報告。管理員可以產生包含預訂資訊的報告，這對於追蹤資源使用情況、評估需求和做出明智的決策非常寶貴。這些報告包含特定的資料字段，使管理員能夠專注於最重要的資訊。
