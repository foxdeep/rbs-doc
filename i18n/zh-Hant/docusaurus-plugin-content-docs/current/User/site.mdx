- 🌍 瀏覽和探索

  進入站點頁面後，您會發現一個使用者友善的介面，其中顯示與所選站點關聯的所有資源的清單。您可以瀏覽此清單以詳細探索資源。每個資源都附有相關信息，例如名稱、描述、可用性以及與其使用相關的任何其他詳細信息。

- 🔎 過濾和排序

  為了使資源發現更加方便，您通常可以使用篩選器和排序選項。這些功能可讓您根據特定條件（例如資源類型、可用性、容量或任何其他相關參數）縮小資源清單的範圍。透過使用篩選器和排序，您可以快速識別最能滿足您需求的資源。

- 🗒️ 資源詳情

  點擊站點頁面中的特定資源可以存取更詳細的資訊。您可以查看有關資源的其他信息，包括與其使用相關的任何特定規則、要求或指南。這可確保您擁有做出明智的預訂決定所需的所有資訊。
