「我的預訂」是用戶儀表板中的一個功能，提供動態且互動式的日曆視圖，展示您所有的預訂。此功能旨在為您提供清晰且有條理的預訂概覽，讓您能夠有效地管理行程。

- 📅 日曆視圖

  日曆提供直觀且有條理的方式來查看您的預訂。每個預訂都顯示在相應的日期上，方便您追蹤和管理行程。

- 🔄 切換視圖

  為了滿足不同的偏好，日曆支持多種視圖，例如月視圖、週視圖、日視圖和列表佈局。您可以輕鬆切換這些視圖，以獲得更廣泛或更詳細的預訂概覽。

- 🔍 按日期範圍篩選

  日曆包含日期範圍篩選功能，允許您將視圖縮小到特定的時間範圍內。這確保您能快速專注於選定範圍內的預訂。

- 🖊️ 管理您的預訂

  日曆上的每個預訂都是互動式的，您可以直接從日曆界面執行特定操作。例如：

  - ❌ 取消預訂

    如果您需要取消預訂，只需點擊該預訂並選擇取消選項。這提供了一種快速且無憂的方式來管理行程變更。

  - 📋 查看詳情

    點擊預訂還可以查看該預訂的詳細信息，包括資源、時間和任何附加備註。

通過這些功能，「我的預訂」日曆確保了管理您的預訂既直觀又高效。
