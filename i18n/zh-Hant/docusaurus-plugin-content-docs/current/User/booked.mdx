成功完成預訂後，您將被引導至“預訂結果頁面”，您將在收到預訂確認訊息。在這裡，您有幾個選擇：

- **✅ 成功**或**❌ 失敗** 確認：

  將清楚地表明您的預訂是否成功或失敗。成功意味著您的預訂已被確認並可啟動流程。

---

<span class="site-user">站點成員</span>的選項：

如果您是站點成員，您將在視窗上看到以下三個按鈕：

- **新增到行事曆**

  點擊「新增到行事曆」，您可以輕鬆將預訂整合到您偏好的日曆應用程式（例如，Google 日曆、iOS 日曆或 Microsoft 日曆）。此功能幫助您保持井然有序，確保您不會忽略任何重要的預訂細節。

- **我的訂單**

  點擊「我的訂單」將引導您進入[後台](../user/dashboard)，特別是[我的訂單](../user/dashboard/my-orders)部分。在這裡，您可以查看和管理您的所有預訂，確保您可以輕鬆存取您的預訂記錄和狀態。

- **返回網站**

  如果您想繼續瀏覽網站或查看更多資源，您可以透過點擊「返回網站」返回網站主頁。

---

<span class="guest">訪客</span>的選項：{' '}

如果您以訪客身分預訂，您**_不會_**在視窗上看到「啟動流程」按鈕。請檢查您的<mark>電子郵件</mark>是否有包含啟動預訂流程連結的訊息。此連結將指導您完成所有必要的步驟以完成您的預訂。
