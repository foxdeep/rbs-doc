歡迎來到顧客(員工)入口，這是您高效資源管理的入口。顧客(員工)入口網站的登入頁面是所有與資源相關的活動的起點。無論您是想要註冊的新用戶、管理員還是準備登入的現有用戶，此頁面都提供無縫且使用者友善的體驗。

- 🔍 搜尋可用站點

  入口的主要功能之一是能夠搜尋可用網站。用戶可以輕鬆發現並瀏覽可用網站的清單。

- 👤 註冊為管理員/用戶

  如果您是系統新手或需要建立帳戶，登入頁面會提供使用者友善的註冊流程。在這裡，您可以根據需要選擇註冊為管理員或使用者。註冊過程非常簡單，讓您可以立即開始資源管理。
