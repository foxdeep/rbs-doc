# Versions

This page provides an overview of the RBS service versions.

## Version 1.0

- Initial release of the RBS service.
- Key features:
  - Basic resource management functionality.
  - Core API support.
  - Compatibility with KH-HR's massage booking requirements.

## Version 1.1

- Key features:
  - Improved time period logic for resource management.
    - Compatibility with Yangxin/CESBG General Affairs lunch booking requirements.
  - Resolved known issues with bug fixes.
