The VIP Reservations feature allows administrators to manage and track reservations for VIP clients. This includes viewing reservation details such as status, ID, item, reservation time, applicant, and email address. Additionally, administrators can create new reservations for VIP clients and export the reservation list as an Excel file.

### Key Features

- **Reservation Status**: Easily identify the current status of a reservation (e.g., booked, pending, or canceled).
- **Detailed Information**: View essential details like reservation ID, item type, time slot, and applicant information.
- **Search and Filter**: Quickly locate specific reservations using the search bar or apply filters for more refined results.
- **Pagination**: Navigate through multiple pages of reservations with ease.
- **Create Reservations**: Administrators can create new reservations for VIP clients directly from the system.

  ![snapshot](/assets/admin/resource-management-vip-reservation-add.png)

- **Export to Excel**: Export the reservation list as an Excel file for reporting or record-keeping purposes.

This feature ensures efficient management of VIP reservations, providing a seamless experience for both administrators and clients.
